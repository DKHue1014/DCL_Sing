  1. 준비: 오후 4시쯤부터 와츠앱 단톡에서 누가 떴다고 외치면 본격 시작 시간
  2. 경로 고정: 디즈니 앱 → 홈화면의 온보드펀(My Plan의 Add Onboard Fun 아님!) → 어트랙션 → 아이언테스트런
  3. 리프레시 패턴: 매시 정각 / 30분에 슬롯 열림, 29분 / 59분에 사람들 취소분이 풀림 → 시간 직전부터 리프레시 반복
  4. 비활성→활성 토글: 비활성 상태면 다시 어트랙션 페이지로 빠져나와 재진입. 활성 보이면 즉시 시간 선택 → 최종 예약 버튼
  5. 튕김 대비: 최종 예약 단계에서 자주 튕김. 한번에 안 된다고 포기 X, 같은 시간대 다시 시도
